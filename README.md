# 📸 Photo Editor using OpenCV & Gradio

## 📌 Project Description

This project is a web-based photo editing application built using OpenCV and Gradio.
It allows users to upload an image, apply various image processing techniques, and download the edited result.
The application provides an interactive interface where users can control image transformations using sliders and dropdown options.

---

## 🛠 Tools Used

* Python
* OpenCV
* Gradio
* NumPy
* Pillow

---

## ✨ Features of the Application

* 📤 Upload Image
* 📏 Resize Image (Width & Height control)
* 🎚 Adjust Brightness
* 🎚 Adjust Contrast
* 🎨 Apply Filters:

  * Grayscale
  * Blur
  * Sharpen
  * Warm Filter
  * Portrait Background Blur
  * Edge Detection *(Extra Feature)*
  * Cartoon Effect *(Extra Feature)*
* 📥 Download Edited Image

---

## ▶️ Steps to Run the Project

### 1. Clone the Repository

```bash
git clone <your-repo-link>
cd photo-editor
```

### 2. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
python app.py
```

### 4. Open in Browser

The application will open automatically in your browser.
